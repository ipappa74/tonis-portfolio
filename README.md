# Portfolio

Moderni portfolio-sivusto HTML:llä, CSS:llä ja JavaScriptilla.

## Ominaisuudet

- vaalea / tumma teema
- aktiivinen navigaatio scrollatessa
- responsiivinen layout
- projektikortit screenshot-kuvilla
- profiilikuva hero-osiossa
- fallbackit puuttuville kuville
- julkaistavissa GitHub Pagesiin
